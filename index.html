Spring-remoting-sec-example

Spring RMI with remote UserDetailsManger to use in client authentication manager.

Download this project as a .zip file Download this project as a tar.gz file

Cloning and building

$ git clone git://github.com/seenukarthi/spring-remoting-sec-example.git
$ cd spring-remoting-sec-example
$ mvn clean install

Running

To run this project open two terminal on project root folder 
Terminal 1 (Server)

$ cd spring-remoting-server
$ mvn clean package tomcat:run -Dmaven.tomcat.port=8081


Terminal 2 (Client)

$ cd spring-remoting-webclient
$ mvn clean package tomcat:run

Open http://localhost:8080/spring-remoting-webclient in the browser and give admin/admin as credientials.
